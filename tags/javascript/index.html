6.2、JavaScript 没有数组数据结构。但提供了一种类数组特性的对象。它把数组的下标转变成字符串，用作其属性。

6.3、JavaScript 中这种类“数组特性的对象”与一般对象的不同点在于：

  1. 此对象继承自Array.prototype，而一般对象继承自Object.prototype；
  2. 此对象拥有一个诡异的length属性，而一般对象没有。
